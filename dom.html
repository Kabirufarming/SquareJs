SquareJs Dom

Dom Css

Gets or sets style values.

  • param: Object element A DOM element
  • returns: Object The style value

Example

squareJs.dom.css($square('#element')[0], {
    'padding': '5px', //gives a margin of 5px
    'background': '#000000' //set a black background color for element
});

Dom Get

Finds DOM elements based on a CSS selector.

  • param: String selector A CSS selector
  • returns: Array The elements

Example

squareJs.dom.get($square('#element'));

Dom FindElement

Does an element satify a selector, based on root element?

  • param: Object element A DOM element
  • param: String selector A CSS selector
  • param: Object root The root DOM element
  • returns: Object The matching DOM element

Example

squareJs.dom.findElement($square('#element'), 'class', $square('body'));
