3 simple steps to squareJs

  1. Create page with HTML5 DOCTYPE (not required).
  2. Include squarejs.min.js
  3. Use markup, as described in the documentation page of this site
<!DOCTYPE html>
<html>
    <head>
	    <script src="path/to/file/squarejs.min.js"></script>
	    <script>
	    	squareJs.events.ready(function () {
	    		$square('#example').html("Welcome to SquareJs!");
	    	});
	    </script>
    </head>
    <body>
    	... page ...
    </body>
</html>
